# Road-Accidents-Analysis

<img width="1599" height="1599" alt="Road_Accidents_graph" src="https://github.com/user-attachments/assets/84cfe14a-16e7-4dbb-bba4-4429817ab375" />

1) Collected and Imported Accident Data
Loaded and examined a road accident dataset using Python libraries such as Pandas and NumPy.
2) Performed Data Cleaning
Handled missing values, removed duplicates, corrected data types, and prepared the dataset for analysis.
3) Conducted Exploratory Data Analysis (EDA)
Explored accident patterns and trends through statistical summaries and visualizations.
4) Analyzed Accident Severity
Investigated the distribution of accident severity levels and identified factors contributing to severe accidents.
5) Studied Time-Based Accident Trends
Examined accident occurrences across different hours, days, months, and seasons to identify peak accident periods.
6) Investigated Environmental Factors
Analyzed how weather conditions, road surface conditions, and lighting conditions affect accident frequency.
7) Performed Location-Based Analysis
Identified accident-prone regions and high-risk locations using geographical data and visual representations.
8) Created Data Visualizations
Developed charts and graphs using Matplotlib and Seaborn to present insights clearly and effectively.
9) Extracted Actionable Insights
Identified major causes and contributing factors behind road accidents to support road safety awareness.
10) Generated Recommendations
Proposed data-driven suggestions for reducing accidents, improving traffic management, and enhancing road safety measures.
